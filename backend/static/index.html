SHT

A tiny content-addressed blob store with named aliases.

Install

This is a placeholder install flow for now. 

curl -fsSL https://shthub.org/bin/install-shh.sh | bash

URLs

  • /u/<user>/<alias>?raw view raw
  • /u/<user>/<alias>?html render to HTML
  • /b/<digest> fetch exact blob
  • /healthz health check

Examples

https://shthub.org/u/benjamin/go.mod?raw
https://shthub.org/u/benjamin/go.mod?html
https://shthub.org/b/nI8PzFREgDLwvTU8c7GOwvmYUXKZkJG_2OiEFvrEpUQA

Client usage

shh up -o go.mod go.mod
shh url go.mod

Uploads a file, records a new version under the alias, and prints shareable links.

Notes

  • Aliases can have multiple versions (append-only list of digests).
  • A digest is the content identifier.
  • Use ?raw for inline viewing.
